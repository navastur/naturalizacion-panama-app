# Sistema de Puntos de Experiencia (XP) - Naturalización Panamá

Este sistema está diseñado para incentivar el estudio constante y premiar la precisión en las respuestas.

| Actividad | XP Base | Bono / Multiplicador |
| :--- | :---: | :--- |
| **Completar un Vídeo de Estudio** | 20 XP | +5 XP si se ve hasta el final sin saltos. |
| **Escuchar un Mini-podcast completo** | 15 XP | +5 XP si se termina el episodio sin pausas largas. |
| **Sesión de Flashcards (Tarjetas)** | 15 XP | +10 XP si se repasan todas las tarjetas del mazo. |
| **Completar un Módulo de Estudio** | 50 XP | +20 XP si es la primera vez que se completa. |
| **Simulacro de Examen (Completo)** | 100 XP | +50 XP por puntaje perfecto (20/20). |
| **Examen de Refuerzo Inteligente** | 120 XP | +20 XP bono por enfrentar puntos débiles (15 preguntas). |
| **Respuesta Correcta (Individual)** | 2 XP | +1 XP por racha de 5 respuestas seguidas. |
| **Dominar una Pregunta (Estado Verde)** | 10 XP | Bono por acertar 3 veces seguidas la misma pregunta. |
| **Entrada Diaria (Login)** | 10 XP | Se multiplica por el número de días de racha (máx x5). |

## Progresión de Niveles (Curva de Aprendizaje)
Para mantener el interés, la dificultad para subir de nivel aumenta progresivamente:

- **Nivel 1 a 5 (Novato):** 500 XP por nivel.
- **Nivel 6 a 15 (Estudiante):** 1,000 XP por nivel.
- **Nivel 16 a 25 (Patriota):** 2,500 XP por nivel.
- **Nivel 26+ (Ciudadano Experto):** 5,000 XP por nivel.

## Multiplicadores de Racha (Streaks)
- **Día 3:** x1.2 XP en todas las actividades durante ese día.
- **Día 7:** x1.5 XP en todas las actividades durante ese día.
- **Día 30:** x2.0 XP en todas las actividades durante ese día + Insignia "Compromiso Nacional".
